Group Projects

These are the work I've done with others
during group projects and game jams.

Dark Escape

This is a 2d roguelike game created within Unity in a module called Technical Games Production (TGP) during second year. The players will have to go through dungeon and defeat the boss to escape it. I worked on the soundtrack composing, audio proggramming, and loot generation of the project.

Tags: Unity, C#, Loot Table Generation, Audio Programming, Agile Development

  • Github Repo

Spirit of Shinobi

This is a 3d topdown 2-player shooter game created within Unity during the Global Game Jam 2022 right after TGP. Each player can shoot black and white shurikens to defeat enemies of the opposite color. I worked the VFX design and the soundtrack composing of the project.

Tags: Unity, C#, VFX Design, Audio Programming, Agile Development

  • Global Game Jam link

Getting into It

This is a 2d platformer Created within Unity during the Staffs jam 2022 during summer. This game is a parody of getting over it. I worked on the soundtrack composing, audio programming, soundtrack composing sand act as the source control manager of the team.

Tags: Unity, C#, Audio Programming, Github Management, Agile Development

  • itch.io link

The Nightmare by Nightfall

This is a 2d side scroller action platformer created with an original engine made using OpenGL during the Studio Sim module during Final Year. The game has a dark gothic theme like Castlevania. I worked on the soundtrack composing, audio & level design of the project.

Tags: OpenGL, C++, Audio Design, Level Design, Agile Development
